## Installation

To get started, install the GSK Icons package:

```bash
npm i @gsk/icons
```
Check out our [GSK Icons installation guide](https://www.untitledui.com/react/docs/icons).


## Using GSK Icons

Import GSK Icons directly into your React project:

```jsx
import ArrowDown from "@gsk/icons/ArrowDown"

<ArrowDown />

// Configure size via `size` prop
<ArrowDown size={16} />

// Configure size via `className`
<ArrowDown className="size-4" />

// Configure color via `color` prop
<ArrowDown color="#000" />

// Configure color via `className`
<ArrowDown className="text-black" />
```

## What are GSK Icons?
